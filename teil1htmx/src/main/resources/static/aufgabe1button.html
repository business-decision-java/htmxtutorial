Aufgabe 1: Erste htmx-Button-Experimente

Ausgangslage: Was passiert wenn Du den Button auf dieser Seite drückst?

  1. Binde nun die htmx-Library in die Seite ein, so dass wir dann den Button mit htmx Attributen dekorieren können.
  2. Im GreetingController findest Du eine Ressource "/clicked". Füge dem Button nun ein möglichst einfaches **hx-get** (ohne weitere Attribute) auf diese Ressource hinzu. Was passiert mit dem Button beim Klicken?
  3. Sorge nun dafür, dass das HTML-Element Button als ganzes ersetzt wird (und nicht nur der Text innerhalb des Buttons).
  4. Nun soll das Resultat der Ressource nicht den Button ändern, sondern als Wert innerhalb des div mit der ID "resultat" angezeigt werden.
  5. Aktuell gilt für den Button der Standard-Trigger "click" zum Auslösen, des hx-get. Stelle das so um, dass es zusätzlich auch bereits ausgelöst wird, sobald die Maus über den Button geschoben wird.
  6. Per default wird der Trigger sofort ausgelöst. Füge eine Verzögerung hinzu, so dass 500ms nachdem die Maus über dem Button ist, das hx-get ausgelöst wird.
